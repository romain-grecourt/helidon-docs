# Helidon-OCI Native Services Integration Guide

---

## Overview

The oci-helidon project contains a compilation of different modules that extend the use of the Helidon framework to allow integration with Native OCI services. This project requires [Helidon 4](https://helidon.io/docs/v4/about/doc_overview) and JDK21 as [prerequisites](https://helidon.io/docs/v4/about/prerequisites). The following are the list of features that are currently offered:
1. Support integration with various OCI native services that includes: 
   * [Splat](./splat.md)
   * [Metrics](./metrics.md)
   * [Secret Service](./secret-service.md)
   * [Lumberjack](./lumberjack.md)
   * [Identity](./identity.md)
2. Libraries that help facilitate handling of various OCI native proprietary capabilities such as:
   * [error-code](./error-code.md)
   * [request-id](./request-id.md)
   * [Jipher for FIPS compliance ](./jipher.md)
