Configuration of Helidon Microprofile Server

Type:
[io.helidon.microprofile.server.Server](https://helidon.io/docs/v4/apidocs/io.helidon.microprofile.server/io/helidon/microprofile/server/Server.html)

This is a standalone configuration type, prefix from configuration root:
`server`

# Configuration options

| key    | type   | default value | description            |
|--------|--------|---------------|------------------------|
| `host` | string |              | Configure listen host. |
| `port` | int    |              | Configure listen port. |
