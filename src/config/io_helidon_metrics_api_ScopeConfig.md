# Configuration options

| key | type | default value | description |
|----|----|----|----|
| `enabled` | boolean | `true` | Whether the scope is enabled. |
| `filter.exclude` | Pattern |  | Regular expression for meter names to exclude. |
| `filter.include` | Pattern |  | Regular expression for meter names to include. |
| `name` | string |  | Name of the scope to which the configuration applies. |
